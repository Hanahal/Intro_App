# Aplicación CRF (Cemented Rockfill)

Esta es una aplicación desarrollada en **Python y Streamlit** para realizar cálculos, análisis y visualización relacionados al diseño de CRF (Cemented Rockfill).  
La aplicación permite cargar datos, mostrar tablas, generar gráficos y procesar resultados geomecánicos.

## Archivos del proyecto
- **app.py** — Código principal de la aplicación.
- **requirements.txt** — Lista de librerías necesarias para ejecutar la app.
- **README.md** — Breve descripción del proyecto.
